# Arc + Finch Studio (Demo)

Static 5-page marketing site inspired by an editorial, quiet-luxury aesthetic:

- `index.html` (Home)
- `services.html`
- `work.html`
- `about.html`
- `contact.html`

## Run Locally

Open `index.html` directly, or (recommended) serve the folder so navigation + transitions behave consistently:

```bash
cd /home/no/Desktop/creativesite
python3 -m http.server 5173
```

Then visit `http://localhost:5173`.

## Notes

- Photos are loaded from Unsplash via `source.unsplash.com` (placeholders for the demo).
- Motion respects `prefers-reduced-motion`.
